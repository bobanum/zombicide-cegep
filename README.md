## Compétences
- Air Bête : Est intimidant. Les Zélèves dans la case à la fin du tour doivent reculer d'une case.
- Inquisiteur : Ne répond aux questions que par d'autre questions.
- Enthousiasme : Les Zélèves vont tourmenter les autres survivants avant de s'occuper de lui.
- Action +1 : Permet d'effectuer une action supplémentaire pendant son tour.
- Team Maker : Permet de créer des équipes de travail de 3 personnes. À la fon du tour, un des Zélèves est éliminé par les 2 autres.
- Organisateur-né : Permet d'échanger jusqu`à 4 Zélèves vers l'avant. À la fin du tour, un Zélève sur 2 est éliminé à cause de la pression.
- 3e année : Le survivant ne voit pas les zélèves de type fatty. Ceux-ci avance de 2 cases dans leur direction. Par contre, il est slippery contre les runners.
- Érudit : +1 sur un jet de dés.
- Sympatique: Permet de relancer un jet de dés.
- Those who can : (Un professeur qui n'est pas un érudit)
- Rassembleur : +1 dé 
- Intransigeant : Les fatties sont automatiquement éliminés par une évaluation. Les walkers subissent un malus de -1.
